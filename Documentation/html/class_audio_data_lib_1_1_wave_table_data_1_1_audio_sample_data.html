Detailed Description

This is AudioData with extra information needed by a synthesizer.

Note that the raw audio data stored here should be mono, and then there is a ChannelType member that is used to indicate the intended purpose of the sample.

◆ Mode

enum AudioDataLib::WaveTableData::AudioSampleData::Mode : uint8_t
Enumerator
NOT_LOOPED 

The sample is not meant to be looped at all. Think of a harp pluck or a piano key strike.

GETS_TRAPPED_IN_LOOP 

After playing the sound that comes before the loop, we then stay in the loop until the loop is terminated. Think of a trumpet or clarinet (with great lung capacity!)

UNUSED 

This is unused as per the sound-font file format documentation.

EXIT_LOOP_ON_RELEASE 

This is just like GETS_TRAPPED_IN_LOOP, except that when the loop is termianted, we're meant to exit at the end of the loop and then play the remainder of the sample that exists after the loop.
